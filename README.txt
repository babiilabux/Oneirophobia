Agir sur les branches git : 
CHANGER DE BRANCHE : checkout [nom de la branche]
CREER UNE BRANCHE : checkout [nom de la nouvelle branche]
AFFICHER LA LISTE DES BRANCHES ACCESSIBLES : git branch
PULL LES ELEMENTS : git pull origin [nom de la branche]



Quand une branche est créée, il faut ajouter le remote à cette branche avant de publier de la branche.
